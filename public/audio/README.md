# PLACEHOLDER AUDIO FILES

This directory contains placeholder audio files for the Valentine's Movie Theatre app.

## Required Files:

1. **projector-loop.mp3**
   - Film projector background sound (looping)
   - Should be a subtle, continuous projector whirring sound
   - Recommended: Low volume, ambient quality

2. **morse-code.mp3**
   - Morse code pattern for Feb 7 game
   - Pattern should encode: "15/10/25"
   - Looping audio

3. **valentine-song.mp3**
   - Your favorite song for the Feb 14 vinyl player
   - Any format supported by Howler.js (mp3, wav, ogg, etc.)
   - Will loop when playing

## How to Add:

Simply replace these placeholder files with your actual audio files, keeping the same filenames.
